# Cost Calculation for LLM Requests

## Parameters

- **Cost per 1000 Tokens (Input)**: $3
- **Cost per 1000 Tokens (Output)**: $15
- **Average Input per Request**: 1000 Tokens
- **Average Output per Request**: 2000 Tokens
- **Total Requests**: 100,000

## Calculation

1. **Total Input Tokens**:
   \[
   \text{Total Input Tokens} = 1000 \times 100000 = 100,000,000 \text{ Tokens}
   \]
2. **Total Output Tokens**:
   \[
   \text{Total Output Tokens} = 2000 \times 100000 = 200,000,000 \text{ Tokens}
   \]
3. **Cost for Input Tokens**:
   \[
   \text{Cost for Input} = \left(\frac{100,000,000}{1000}\right) \times 3 = 300,000 \text{ USD}
   \]
4. **Cost for Output Tokens**:
   \[
   \text{Cost for Output} = \left(\frac{200,000,000}{1000}\right) \times 15 = 3,000,000 \text{ USD}
   \]
5. **Total Cost**:
   \[
   \text{Total Cost} = 300,000 + 3,000,000 = 3,300,000 \text{ USD}
   \]

## Summary

- **Total Cost for 100,000 Requests**: **$3,300,000**
